# Nexa

Futuristic bilingual AI web app concept built with Next.js, TailwindCSS, Framer Motion, and Zustand.

## Run

```bash
npm install
npm run dev
```

## Features

- Cinematic Arabic intro with animated CTA
- Local login and personalized greeting
- Conversational UI with typing indicator and lightweight AI persona
- Auto split-screen preview when code blocks are detected
- Mobile tab switch for chat/preview
- Local chat history persistence + clear chat
- Dark/light mode toggle
- CI workflow for production build + optional Vercel deploy
